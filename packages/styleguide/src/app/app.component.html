Deprecation notice

As of October 2022 we deprecated the use of Angular as a frontend solution. This means that new frontend solutions should always be built using React.js.

Existing projects can still do a one-time upgrade to the latest version of Angular (and accompanying components), but should migrate, as described in the Digipolis Antwerpen Application Stack (DAAS).

Always check https://digitalehuisstijl.antwerpen.be for the latest news and guidelines concerning Antwerp's digital branding guidelines.
